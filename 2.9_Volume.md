# Volume
## emptyDir
```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx-date
spec:
  containers:
    - image: nginx:1.20
      name: web
      volumeMounts:
        - mountPath: /usr/share/nginx/html
          name: html
          readOnly: true
    - image: artemshestakov/date
      name: generator
      volumeMounts:
        - mountPath: /var/html
          name: html
      ports:
        - name: http
          containerPort: 80
          protocol: TCP
  volumes:
    - name: html
      emptyDir: {}
```
Create `emptyDir` in memory not in the host of file system:
```yaml
volumes:
  - name: html
    emptyDir:
      medium: Memory
```

## gitRepo
```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx-git
spec:
  containers:
    - image: nginx:1.20
      name: web
      volumeMounts:
        - mountPath: /usr/share/nginx/html
          name: git
          readOnly: true
      ports:
        - name: http
          containerPort: 80
          protocol: TCP
  volumes:
    - name: git
      gitRepo:
        repository: https://github.com/cxxr/example-website-repo.git
        revision: master
        directory: "."
```

## nfs
```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nfs-pod
spec:
  containers:
    - name: date-generator
      image: artemshestakov/date:1.0
      volumeMounts:
        - mountPath: /var/html
          name: nfs-volume
  volumes:
    - name: nfs-volume
      nfs:
        server: 192.168.56.10
        path: /var/nfs/general
```

## PersistentVolume
```yaml
apiVersion: v1
kind: PersistentVolume
metadata:
  name: nfs
spec:
  accessModes:
    - ReadWriteOnce
    - ReadOnlyMany
  capacity:
    storage: 1Gi
  nfs:
    server: 192.168.56.10
    path: /var/nfs/general
  persistentVolumeReclaimPolicy: Retain
```

## PersistentVolumeClaim
```yaml
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: pvc-nfs-rwo
spec:
  accessModes:
    - ReadWriteOnce
  resources:
    requests:
      storage: 1Gi
  storageClassName: ""
```
